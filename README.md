## Healthcare-Analytics-Project
Healthcare Waiting List Operational Analysis - Ireland 2025

## Business Problem
Hospitals want to analyse patient data to identify patterns and improve healthcare outcomes.

## Objective
To explore healthcare data and build a predictive model for patient risk assessment.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- SQL

## Project Steps
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature engineering
4. Model building
5. Evaluation and insights

## Results
Model performance and key insights will be documented here.

## Business Impact
This project demonstrates how data analytics can support healthcare decision-making.
